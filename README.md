## 🎯 核心玩法

- **单发精准射击**：按键边沿检测，每次按下只发射一颗子弹；获得速射道具后冷却降至 5 帧（12发/秒）
- **固定关敌人数**：每关敌人数 = 关卡数 + 1（上限 6 辆），消灭全部敌人进入下一关
- **关卡递增难度**：敌人数递增 + AI 行为进化 + 随机砖块密度递增

## 操作

| 操作 | 键盘 | 触屏 |
|------|------|------|
| 移动 | `↑↓←→` 或 `WASD` | 触摸坦克前方方向 |
| 射击 | `空格`（单发模式） | 触摸右下角 40% 区域 |
| 暂停 | `P` | — |
| 重新开始 | `R` | 点击按钮 |

## 技术架构

```
Tank-Mini-Game/
├── index.html          # 主页面，按依赖顺序加载 JS/CSS 模块
├── css/
│   └── style.css       # 全部样式（覆盖层 / 动画 / HUD / 响应式）
├── js/
│   ├── constants.js    # 游戏常量 + 5 套地图模板 + 道具定义 + AI 参数
│   ├── audio.js        # 8 位音效合成（Web Audio API）
│   ├── utils.js        # 工具函数
│   ├── particles.js    # 爆炸粒子特效
│   ├── powerups.js     # 道具系统
│   ├── bullet.js       # 子弹类
│   ├── tank.js         # 坦克类（移动 / 射击 / AI）
│   ├── map.js          # 地图构建 / 敌人生成
│   ├── input.js        # 键盘 + 触屏输入
│   ├── game.js         # 游戏核心循环
│   └── main.js         # 主入口
├── README.md
└── .gitignore
```

- **Game 全局命名空间** + IIFE 隔离模式，支持 `file://` 协议直接打开
- 零外部依赖，纯前端实现

## 快速开始

直接用浏览器打开 `index.html` 即可：

```bash
start index.html   # Windows
open index.html    # macOS
```
