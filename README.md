# 坦克大战 (Tank Battle)

> 纯 HTML5 Canvas 实现的经典坦克大战网页游戏，键盘 + 触屏双模操控，零外部依赖。

## 特性

### 核心玩法
- **单发精准射击**：按键边沿检测，每次按下只发射一颗子弹；获得速射道具后冷却降至 5 帧（12发/秒）
- **固定关敌人数**：每关敌人数 = 关卡数 + 1（上限 6 辆），消灭全部敌人进入下一关
- **关卡递增难度**：敌人数递增 + AI 行为进化 + 随机砖块密度递增

### 五种地图模板（按关卡轮转）
| 模板 | 风格 | 特点 |
|------|------|------|
| 0 - 对称象限 | 经典对称 | 四象限砖钢混搭 + 中央钢墙堡垒 |
| 1 - 水平走廊 | 横向通道 | 四排水平墙带三组三格宽过道 + 钢墙加固 |
| 2 - 竞技场环形 | 围合战场 | 外圈砖墙带四门入口（侧门三格宽）+ 四角钢柱 + 内部散砖 |
| 3 - 要塞十字阵 | 中央要塞 | 十字钢墙分割（竖墙留两处双格缺口）+ 水平砖墙带钢柱 |
| 4 - 散落堡垒群 | 开放地形 | 八组 2x2 钢砖堡垒 + 中央散砖 |

- 随机散落砖块密度随关卡递增（2.5% + 0.8%/关），地图通透性大幅提升
- 出生区（顶部 3 行 + 底部 3 行）强制清空，确保开局安全
- 钢墙（银色）不可摧毁，砖墙（棕色）可被子弹破坏
- 通道宽度 ≥ 3 格，坦克（26px）可顺畅通过，不卡墙

### 两种 AI 行为模式

| 模式 | 头顶标记 | 行为 | 射击概率 |
|------|----------|------|----------|
| **巡逻** | 蓝点 | 水平/垂直主轴移动，撞墙反转；**反弹 4 次自动换轴防钢墙卡死**；150px 内感知加倍射速，不干扰移动方向 | 2.0%/帧（感知 4%） |
| **守卫** | 琥珀 | 出生点 5 格警戒圈，越界自动回退；圈内巡逻，稳健防御 | 2.5%/帧 |

- **巡逻感知机制**：玩家进入 150px 范围时，巡逻坦克射速翻倍（0.02→0.04）；不改变移动方向（避免卡住感）
- **反弹计数防卡死**：巡逻撞墙反弹 4 次后强制切换巡逻主轴，杜绝在钢墙间来回弹跳
- **定期换轴**：每 3-5 秒有 25% 概率随机切换巡逻主轴，协同防拥堵
- **守卫要塞机制**：警戒半径 5 格（150px），射击概率 2.5%/帧，后期关卡守卫占主力构成要塞防御网
- AI 模式按关卡分配：1-2 关全巡逻 → 3-4 关 20% 守卫 → 5-6 关 40% 守卫 → 7+ 关 70% 守卫
- **卡住突围**：连续 40 帧（0.67s）未位移时随机探索四个方向，自动切换巡逻主轴
- **HUD 实时显示**：左上角第二行展示巡逻/守卫存活数

### 四种道具系统

| 道具 | 图标 | 效果 | 持续时间 | 掉落率 |
|------|------|------|----------|--------|
| 护盾 | 蓝色 | 免疫一次伤害 | 7.5 秒 | 30% |
| 加速 | 橙色 | 速度 1.2→1.8 | 7.5 秒 | 30% |
| 速射 | 红/火 | 冷却 5 帧（12发/秒），火力压制 | 7.5 秒 | 30% |
| 生命+1 | 绿色 | 立即增加一条命 | 即时 | 10% |

- 击毁敌人有 15% 概率掉落道具
- 道具智能生成：避开墙壁 + 8 次偏移尝试，确保不卡墙
- 生命+1 即时生效不占用道具槽，上限 99 条

### 移动物理
- **刚性停止**：撞墙即停，无自动贴墙滑动——靠玩家精确操作通过狭窄通道
- **分离碰撞箱**：视觉尺寸 26px，碰撞箱 24px（居中缩进 1px 缓冲）
- **分轴碰撞检测**（X/Y 独立回退）精准判定
- **网格轻度吸附**：仅小偏差（≤12px）时校正到格点中心，消除浮点卡角
- **AI 反弹计数**：巡逻坦克撞墙反弹 4 次自动换轴，杜绝钢墙弹跳循环
- **边界安全**：全程检测画布边界，防止越界

### 视觉 / 音效
- 8-bit 风格音效（Web Audio API OscillatorNode）：射击、爆炸、受击、通关、游戏结束
- 爆炸粒子系统（扩散 + 减速 + 透明度渐隐）
- 屏幕震动（受击时）、全屏闪白（护盾触发）、无敌闪烁（重生后）
- 道具脉冲光圈动画、关卡过渡缩放淡出
- 敌坦头顶 AI 模式呼吸光晕指示灯

## 操作

| 操作 | 键盘 | 触屏 |
|------|------|------|
| 移动 | `↑↓←→` 或 `WASD` | 触摸坦克前方方向 |
| 射击 | `空格`（单发模式） | 触摸右下角 40% 区域 |
| 暂停 | `P` | — |
| 重新开始 | `R` | 点击按钮 |

## 技术架构

```
tank-battle/
├── index.html          # 主页面，按依赖顺序加载 11 个 JS + 1 个 CSS 模块
├── css/
│   └── style.css       # 全部样式（覆盖层 / 动画 / HUD / 响应式）
├── js/
│   ├── constants.js    # 游戏常量 + 5 套地图模板函数 + 道具定义 + AI 参数
│   ├── audio.js        # 8 位音效合成（Web Audio API）
│   ├── utils.js        # 工具函数（AABB 碰撞 / 颜色转换 / 颜色生成）
│   ├── particles.js    # 爆炸粒子特效
│   ├── powerups.js     # 道具生成（墙碰撞安全检测）/ 激活 / 取消
│   ├── bullet.js       # 子弹类（拖尾 + 光晕 + 墙碰撞销毁）
│   ├── tank.js         # 坦克类（贴墙滑动移动 / 射击 / 伤害 / 双模式 AI / 渲染）
│   ├── map.js          # 地图构建 / 敌人生成（含偏移防重叠）/ AI 分配
│   ├── input.js        # 键盘 + 触屏输入（含单发边沿检测）
│   ├── game.js         # 游戏核心（状态管理 / 更新循环 / 渲染 / HUD）
│   └── main.js         # 主入口（DOM 引用 / 事件绑定 / 启动）
├── README.md
└── .gitignore
```

- **Game 全局命名空间** + IIFE 隔离模式，支持 `file://` 协议直接打开
- 兼容 IE 语法（`var` + 普通 `function`），无需构建工具
- 严格按依赖顺序加载（见 `index.html` 底部注释）

## 快速开始

直接用浏览器打开 `index.html` 即可：

```bash
# Windows
start index.html

# macOS
open index.html

# 或用 Python HTTP 服务器
python -m http.server
```

## 参数调优

在 `js/constants.js` 中可调整所有游戏参数（最终版）：

| 常量 | 默认值 | 说明 |
|------|--------|------|
| `PLAYER_SPEED` | 1.2 | 玩家移动速度（72px/s） |
| `PLAYER_SPEED_BOOST` | 1.8 | 加速道具生效时速度（108px/s） |
| `ENEMY_SPEED` | 0.5 | 敌人移动速度（30px/s） |
| `BULLET_SPEED` | 5 | 玩家子弹速度 |
| `ENEMY_BULLET_SPEED` | 1.5 | 敌人子弹速度 |
| `PLAYER_FIRE_COOLDOWN` | 15 | 玩家射击冷却（0.25s） |
| `PLAYER_RAPID_COOLDOWN` | 5 | 速射道具冷却（12发/秒） |
| `ENEMY_FIRE_COOLDOWN` | 100 | 敌人射击冷却（1.67s） |
| `MAX_ENEMIES_CAP` | 6 | 单关最大敌坦数 |
| `BASE_ENEMY_COUNT` | 2 | 第 1 关基础敌人数（每关 +1） |
| `AI_PATROL_SHOOT_CHANCE` | 0.02 | 巡逻基础射击概率/帧 |
| `AI_PATROL_AWARE_RANGE` | 150 | 巡逻感知距离（5格） |
| `AI_PATROL_AWARE_SHOOT_MULT` | 2 | 感知时射击倍数（0.02×2=0.04） |
| `AI_GUARD_SHOOT_CHANCE` | 0.025 | 守卫射击概率/帧 |
| `AI_GUARD_RANGE` | 150 | 守卫警戒半径（5格） |
| `POWERUP_DROP_CHANCE` | 0.15 | 道具掉落概率 |
| `POWERUP_DURATION` | 450 | 道具持续帧数（7.5s @ 60fps） |
