# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 和 JavaScript 开发的经典坦克大战网页游戏。

## 🎯 游戏特色

- 🕹️ **经典玩法** — 重温经典坦克大战的乐趣
- 🎨 **精美画面** — Canvas 绘制，粒子爆炸特效，屏幕震动
- 📱 **触屏支持** — 支持手机和平板触屏操作
- 🧠 **智能 AI** — 敌方坦克自动寻路和射击
- 📊 **关卡递增** — 随关卡增加敌人数和难度
- 💥 **特效系统** — 爆炸粒子、弹道拖尾、无敌护盾闪烁
- ⌨️ **键盘 + 触屏** — 双重操控方式

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| `↑↓←→` 或 `WASD` | 移动坦克 |
| `空格键` | 发射子弹 |
| `P` | 暂停/继续 |
| `R` | 重新开始 |

### 触屏操作
- 在屏幕上拖动来控制移动方向
- 点击右下角区域开火

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可开始游戏！

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/tank-battle.git
cd tank-battle
# 用浏览器打开
open index.html  # macOS
start index.html # Windows
```

## 🛠️ 技术栈

- 纯 HTML5
- CSS3 (动画与样式)
- JavaScript (ES6+)
- Canvas API (游戏渲染)

## 📝 游戏规则

1. 控制你的绿色坦克消灭所有红色敌方坦克
2. 消灭所有敌人进入下一关
3. 被敌人子弹击中失去一条命
4. 砖墙可被子弹摧毁，钢墙不可摧毁
5. 生命耗尽游戏结束
